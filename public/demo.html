✓
Cross-case pattern detection — flags linked indicators across cases without any API call
✓
Session-cached — generates once per session, never repeats unnecessarily
✓
Supervisor wellness alert — weekly check-in, alerts supervisor if caseworker scores are consistently low

Step 2 of 7

Live CTDC gap detection with trauma-informed tips

As the caseworker types, TRACE checks notes against all 8 CTDC indicator categories in real time. Gaps show trauma-informed phrasing guidance — teaching best practice during the interview, not in a manual.

✓
Fully offline — client-side keyword matching, no API call required
✓
8 CTDC categories — recruitment, documents, movement, wages, exploitation, control, family, duration
✓
Hausa, French, Arabic — notes accepted in any language; translate & structure with one tap

Step 5 of 7

Safe info card — designed for low-literacy contexts

Over 82% of displaced persons in Chad have little or no formal education. The safe info card shows emergency contacts as large, icon-anchored numbers — readable by someone with zero literacy. One tap plays it aloud in French.

✓
Audio playback — Web Speech API reads rights and contacts aloud in French at a slow, clear pace
✓
Icon-anchored numbers — phone, police, shelter — universally understood without text
✓
Caseworker-mediated — survivors never touch the app; caseworker controls delivery

Step 7 of 7

Caseworker wellbeing — because the human matters too

Vicarious trauma and burnout are why experienced anti-trafficking caseworkers leave the field. TRACE checks in weekly — three trauma-informed questions, answers stored only on-device. Supervisors are alerted if scores are consistently low.

✓
Once per week — ISO week gating, never intrusive, skippable
✓
Privacy-first — data stays on device; one-tap screen blur hides case data from bystanders
✓
Real resources — IOM Staff Wellbeing and Headington Institute linked directly
